# Brand Foundation Workbook v1.0

## Goal

Define the strategic decisions that control your brand identity before choosing colors, fonts, icons, or other visual assets. Visual identity without strategy is decoration wearing a business hat.

## 5. Translate Strategy Into Visual Direction

### Color palette logic

Start with emotional intent, not favorite colors.

| Role | Purpose | Notes |
|---|---|---|
| Primary color | Main recognition color | Should match brand personality |
| Secondary color | Supports primary color | Should not compete |
| Accent color | Used for calls to action or emphasis | Use sparingly |
| Neutral dark | Text, contrast, structure | Usually not pure black |
| Neutral light | Backgrounds, space | Usually not pure white |
| Status colors | Success, warning, error | Needed for digital products |

**Color words:**
- Should feel like: `____________________________`
- Should avoid feeling like: `____________________________`

### Typography logic

Pick type based on behavior, not vibes.

| Font role | Use | Personality |
|---|---|---|
| Display / headline | Hero sections, big titles | Most expressive |
| Body | Paragraphs, UI text | Most readable |
| Accent | Labels, quotes, badges | Optional |

## 6. Build the Brand System

### Minimum viable brand kit

Create these first:

1. Logo lockup
2. Primary color palette
3. Typography scale
4. Icon style
5. Social profile image
6. Website hero direction
7. Button and UI style
8. Photography or illustration rules
9. Voice examples
10. Usage rules

### Brand test

A brand direction is working if:

- It is recognizable without the logo.
- It attracts the intended audience.
- It rejects the wrong audience.
- It is usable across digital, print, and social.
- It can produce 20 assets without falling apart.
- It feels distinct from competitors.
