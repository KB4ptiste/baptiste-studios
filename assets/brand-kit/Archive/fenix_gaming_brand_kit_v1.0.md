# Fenix Gaming Brand Kit v1.0

## Goal

Create a practical brand system for **Fenix Gaming**, a strategy-gaming companion brand for PC and Mac players. The system should feel **elegant, bold, playful, and edgy**, without collapsing into the usual neon-gamer landfill. Tiny miracle.

---

## 1. Refined Positioning

### Original direction

For PC & Mac gamers, Fenix Gaming helps them get the most out of strategy games by creating companion web apps and strategy guides, unlike wikis or fan pages that do not create usable content.

### Recommended positioning

> For PC and Mac strategy gamers, **Fenix Gaming** helps players master complex strategy games with focused companion web apps and practical strategy guides, unlike wikis and fan pages that bury useful decisions under trivia.

### Short positioning line

> **Sharper tools for deeper strategy.**

### Brand promise

Fenix Gaming turns complex strategy-game information into usable decisions: builds, counters, maps, timers, faction breakdowns, calculators, and decision guides.

## 3. Color System

### Recommended primary shade

**Primary brand purple: Fenix Violet `#4B2A7B`**

Use this as the core identity purple for logo, major brand surfaces, section headers, gradients, and recognition moments. It is dark enough to feel premium and strategic, but still visibly purple instead of disappearing into black-gray mush.

### Interaction purple

**Arcane Purple `#7C3AED`**

Use this for buttons, selected states, links, active navigation, and important UI actions. The primary purple is too dark to carry every interaction on a dark interface, because apparently physics still applies to branding.

### Palette

| Token | Hex | Role | Usage |
|---|---:|---|---|
| Void Plum | `#17111F` | Primary dark background | App background, website hero, dark cards |
| Night Ash | `#241A33` | Secondary dark surface | Cards, panels, menus |
| Fenix Violet | `#4B2A7B` | Core brand purple | Logo, hero graphics, section branding |
| Arcane Purple | `#7C3AED` | Interactive purple | Buttons, active nav, hover states |
| Rift Lilac | `#A78BFA` | Highlight purple | Glows, badges, small accents |
| Ember Gold | `#F59E0B` | Fenix accent | Rare CTAs, warnings, premium callouts |
| Ash Gray | `#B9B4C7` | Secondary text | Descriptions, metadata, captions |
| Bone White | `#F4F0FF` | Primary text on dark | Headlines, body text on dark |
| Light Fog | `#F7F3FF` | Light background | Docs, guides, long-form reading areas |
| Rune Cyan | `#22D3EE` | System/data accent | Charts, stats, timers, info states |

### Contrast checks

| Pair | Contrast ratio | Verdict |
|---|---:|---|
| Bone White on Void Plum | 16.50:1 | Excellent |
| Ash Gray on Void Plum | 9.17:1 | Excellent |
| Bone White on Arcane Purple | 5.09:1 | Good for normal text |
| Fenix Violet on Light Fog | 10.03:1 | Excellent |
| Rift Lilac on Light Fog | 2.49:1 | Do not use for body text |

### Color usage rule

Use this distribution:

```text
70% dark neutrals
20% purple system
7% lilac/cyan highlights
3% ember gold
```

The ember color should feel like a reward, not like a fast-food button screaming for attention.

## 4. Typography System

## Display Typeface

### Wak

Use **Wak** for display moments only:

- Logo wordmark exploration
- Hero headlines
- Major campaign graphics
- Short labels or faction-style titles
- 1 to 6 word emphasis moments

Avoid using Wak for long strategy guides, tables, dashboards, stats, or body text. It has enough personality to be useful and enough personality to become a nuisance if overused.

### Recommended Wak usage

| Use | Weight |
|---|---|
| Logo / hero | Wak Black or Extra Bold |
| Section titles | Wak Bold |
| Short labels | Wak Medium |
| Decorative text | Wak Book or Light, sparingly |

---

## 5. Font Pairing Options

### Option A: Best practical system

| Role | Font |
|---|---|
| Display | Wak Bold / Black |
| Body + UI | Inter |
| Code/data labels | IBM Plex Mono or Space Mono |

**Why:** Inter is clean and screen-focused, which makes it a strong counterweight to Wak’s expressive display style.

### Option B: More technical/editorial

| Role | Font |
|---|---|
| Display | Wak Extra Bold |
| Body + UI | IBM Plex Sans |
| Data/mono | IBM Plex Mono |

**Why:** IBM Plex Sans feels precise and slightly engineered, which fits strategy tools and guides.

### Option C: More gamer/futurist

| Role | Font |
|---|---|
| Display | Wak Black |
| Subheads | Space Grotesk |
| Body | Inter |

**Why:** Space Grotesk adds a subtle sci-fi edge, but using it for all body copy may make the whole brand feel too trendy.

### My recommendation

Use:

```text
Display: Wak Bold / Black
Body/UI: Inter
Data/technical: IBM Plex Mono
```

This gives you personality up top, readability in the product, and credibility in guides/tools.

## 7. Icon Style

### Recommended icon direction

| Rule | Direction |
|---|---|
| Style | Stroke icons with selective filled details |
| Shape language | Angular with slight corner rounding |
| Stroke weight | 2px at 24px size |
| Corners | 2px to 4px radius |
| Perspective | Flat, no fake 3D |
| Motifs | Phoenix feather, tactical grid, flame shard, compass, rune, faction marker |
| Animation | Small hover pulses, glow states, selected-state flares |

### Icon principle

Icons should look like a **strategy interface**, not fantasy clip art. Use flame and phoenix references as geometry, not literal cartoon birds everywhere. The logo does not need to flap at people. Humanity has suffered enough.

## 8. Logo / Mark Direction

### Recommended direction

Start with a **combination mark**:

1. Wordmark: `Fenix Gaming`
2. Symbol: abstract phoenix feather / flame shard / tactical chevron
3. Optional monogram: `FG`

### Why

A combination mark gives you flexibility:

- Full logo for website header
- Symbol for favicon/app icon
- Monogram for social avatar
- Wordmark for guides and overlays

### Symbol ideas

| Concept | Description |
|---|---|
| Flame shard | A sharp purple flame shape with ember notch |
| Phoenix feather | One angular feather doubling as a cursor or blade |
| Strategy sigil | A diamond/grid mark with phoenix-inspired negative space |
| FG monogram | Interlocked F/G with flame cutout |

Best starting concept:

> **Angular phoenix feather + tactical grid motif**

It connects Fenix, strategy, and digital tools without becoming another bird logo from the branding swamp.

## Verdict

The strongest direction is:

```text
Dark tactical violet brand system
Wak for expressive display
Inter for UI and guides
IBM Plex Mono for data-heavy strategy tools
Angular phoenix-feather icon language
Purple-first palette with rare ember accents
```

This gives Fenix Gaming a brand that can scale across guides, tools, dashboards, social posts, overlays, and future premium products without looking like a wiki wearing eyeliner.
