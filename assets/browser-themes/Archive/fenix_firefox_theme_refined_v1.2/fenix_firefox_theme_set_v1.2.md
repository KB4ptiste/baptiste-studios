# Fenix Firefox Theme Set v1.2

## Change summary

Version 1.2 keeps the three approved themes from v1.1 and adds one new dark variant:

1. Fenix Night Command
2. Fenix Smoked Command
3. Fenix Astral Rare
4. Fenix Ember Rare

## New theme: Fenix Ember Rare

Fenix Ember Rare uses the same structural logic as Astral Rare, but replaces the Astral Magenta accent with Night Command's Ember Gold `#F59E0B`.

### Key color mapping

| Firefox color key | Value | Role |
|---|---:|---|
| `frame` | `#150A24` | Black Violet browser frame |
| `frame_inactive` | `#12091C` | Violet Well inactive/sunken chrome |
| `toolbar` | `#1B1226` | Night Violet toolbar |
| `toolbar_field` | `#2B2110` | Ember Well URL field |
| `toolbar_field_border_focus` | `#F59E0B` | Ember Gold focus/accent |
| `tab_selected` | `#27173A` | Arcane Panel selected tab |
| `tab_line` | `#F59E0B` | Active tab accent |
| `icons_attention` | `#F59E0B` | Attention icons / rare emphasis |
| `popup_highlight` | `#2B2110` | Ember-highlighted popup selection |

## Usage verdict

Use Fenix Ember Rare when the theme needs a warmer premium signal than Astral Rare. Astral Rare reads mystical/factional. Ember Rare reads premium, decisive, and slightly more tactical.

## Firefox testing

1. Open Firefox.
2. Go to `about:debugging#/runtime/this-firefox`.
3. Select **Load Temporary Add-on...**.
4. Open one folder under `themes/`.
5. Select `manifest.json`.
6. Compare frame, toolbar, tab line, URL field, popups, and attention icons.
