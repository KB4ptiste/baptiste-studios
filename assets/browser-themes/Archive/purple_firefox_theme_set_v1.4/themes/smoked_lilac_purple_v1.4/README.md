# Smoked Lilac Purple Firefox Theme v1.4

A static Firefox theme using project color tokens without brand-specific naming.

## Visible palette

| Role | Color | Hex |
|---|---|---:|
| Browser frame | Smoked Lilac | `#DED2E8` |
| Toolbar | Rune Mist | `#E9E0F1` |
| Selected tab | High Rune | `#F3EEF8` |
| Active-tab accent | Command Violet | `#6D28D9` |
| Text | Void Ink | `#17111F` |

## AMO submission note

This theme has no scripts, no permissions, no remote resources, and no data transmission.
