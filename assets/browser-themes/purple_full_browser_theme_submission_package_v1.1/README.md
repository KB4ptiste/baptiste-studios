# Purple Full Browser Theme Submission Package v1.1

This unified archive contains the four purple browser themes for Firefox AMO, Chrome Web Store, Microsoft Edge Add-ons, Opera GX, DuckDuckGo recipes, and Safari macOS tint recipes.

Start here:

```text
docs/purple_full_browser_theme_submission_guide_v1.1.md
```

For the fastest upload workflow, use:

```text
submission_uploads/
```

## Included platforms

```text
firefox_amo/                 # Firefox AMO-ready sources, metadata, previews, icons
chrome_web_store/            # Chrome Web Store theme packages
microsoft_edge_addons/       # Microsoft Edge Add-ons theme packages
opera_gx_mods/               # Opera GX Mod packages
duckduckgo/                  # Appearance recipes only
safari_macos/                # Site tint / web app recipes only
submission_uploads/          # Consolidated upload-ready files
all_previews/                # Consolidated preview images
shared/                      # Shared icons and preview assets
docs/                        # Submission guides and index
```

## Themes

- Black Violet Purple
- Smoked Lilac Purple
- Black Violet Magenta
- Black Violet Orange

## Version note

This is the unified package version. Firefox remains v1.4 internally because that package was already ready for AMO. Chrome, Edge, and Opera GX remain v1.0 internally because this merge did not change their manifests.
