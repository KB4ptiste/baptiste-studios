# Smoked Lilac Purple Chrome Theme v1.0

Color-only Chromium theme package for Chrome.

## Test locally

1. Open the browser extensions page.
2. Enable Developer Mode.
3. Choose **Load unpacked**.
4. Select this folder.

## Public submission

Upload the source ZIP from the sibling `source` folder. Do not upload the parent folder; the manifest must be at the ZIP root, because apparently archives are where software stores its grievances.
