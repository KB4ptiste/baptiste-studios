# Purple Full Browser Theme Submission Index v1.1

This unified package adds the Firefox AMO-ready theme set into the broader browser package.

## Themes

| Theme | Core read | Best public use |
|---|---|---|
| Black Violet Purple | Dark purple with purple accent | Primary dark theme |
| Smoked Lilac Purple | Light smoky lilac with purple accent | Primary light theme |
| Black Violet Magenta | Dark purple with magenta accent | Specialized vivid/dark variant |
| Black Violet Orange | Dark purple with orange-gold accent | Premium warm dark variant |

## Version note

This is a unified package version. Platform package internals keep their existing component versions:

- Firefox AMO theme set: v1.4
- Chrome/Edge/Opera GX port set: v1.0
- Unified full-browser archive: v1.1

That avoids unnecessary manifest churn while still giving you one coherent submission archive.
