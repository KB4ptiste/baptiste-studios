# Black Violet Orange Firefox Theme v1.4

A static Firefox theme using project color tokens without brand-specific naming.

## Visible palette

| Role | Color | Hex |
|---|---|---:|
| Browser frame | Black Violet | `#150A24` |
| Toolbar | Night Violet | `#1B1226` |
| Selected tab | Arcane Panel | `#27173A` |
| Active-tab accent | Ember Gold | `#F59E0B` |
| Text | Rune Lilac | `#D6C8EA` |

## AMO submission note

This theme has no scripts, no permissions, no remote resources, and no data transmission.
