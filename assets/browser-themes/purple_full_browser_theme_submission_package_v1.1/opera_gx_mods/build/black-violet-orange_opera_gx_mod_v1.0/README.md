# Black Violet Orange Opera GX Mod v1.0

This is an Opera GX Mod package with theme colors and static wallpapers.

## Local test

1. Open `opera:extensions`.
2. Enable Developer Mode.
3. Click **Load unpacked**.
4. Select this folder.
5. Open `opera:mods` and enable the mod.

## Store submission

Upload the ZIP from `opera_gx_mods/source/` through GX.create / GX.store. The manifest is based on Opera GX Mods documentation and includes both light and dark theme definitions.
